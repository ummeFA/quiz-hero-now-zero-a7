What are the differences between map, filter, foreach and find?

► Map: It runs the same code on every element of an array, returning a new array.
► Filter: It checks if every element fulfils some fixed requirements and also returns a new array.
► Foreach: It executes the same code on every element of an array like map, but it does not change the array. Instead it returns undefined.
► Find: It is used when we need a single element from an array and returns it.

What are the differences between local storage, section storage and cookies?

► Local Storage: It helps to store data in clients' computers where data is stored until the user manually removes them.
► Session Storage: Unlike local storage, it cannot store data for a long time. When the current tab is closed, data is also removed.
► Cookies: It is used to save data that are not sensitive. It can save data upto centain time.
